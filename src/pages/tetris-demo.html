1. Full Tetris Box Component

The <tetris-box> component displays multiple tetris pieces with collision detection:

Usage:

<tetris-box 
  statement="Your text here"
  piece-count="21"
  img-path="/img">
</tetris-box>

2. Single Tetris Piece Component

The <tetris-piece> component displays a single random piece:

Usage:

<tetris-piece img-path="/img"></tetris-piece>

3. Specific Pieces with Rotation

You can specify which piece and rotation:

Usage:

<tetris-piece 
  img-path="/img" 
  piece="i" 
  rotation="90deg">
</tetris-piece>

5. API Reference

<tetris-box> Attributes:

  • statement - HTML content to display as centered text
  • piece-count - Number of pieces to generate (default: 21)
  • img-path - Path to tetris SVG images (default: /img)

<tetris-piece> Attributes:

  • piece - Specific piece name: i, o, t, s, z, j, l (default: random)
  • rotation - CSS rotation value (e.g., "90deg", "180deg")
  • block-size - Size of each block in pixels (default: 25)
  • img-path - Path to tetris SVG images (default: /img)

JavaScript API:

// Get reference to component
const tetrisBox = document.querySelector('tetris-box');

// Refresh/regenerate pieces
tetrisBox.refresh();
